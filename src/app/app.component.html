ghost-writers
augmented reality meets street art
Ghost Writers uses Wikitude's image recognition software to create a mobile augmented reality graffiti application that allows users to post and create their own virtual art, anywhere in the world.

TechStack: Angular 2, Ionic 2, Wikitude, bCrypt, AWS, MongoDB, Node, Express, Google Maps API, FileStack.
